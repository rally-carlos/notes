## Okta
```sh
rally-okta -P  # Configure SSO password for OKTA auth
rally-okta -B. # Bootstrap OKTA configuration
```

## AWS Profiles
```sh
rally-okta -w  # Write AWS Credentials for all profiles
rally-okta -s # View session expiration
```
